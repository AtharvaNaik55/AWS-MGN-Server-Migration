# AWS-MGN-Server-Migration
Production-grade server migration using AWS Application Migration Service (AWS MGN), implementing continuous block-level replication, test launch, validation, and production cutover. Integrated Amazon EC2, VPC, ELB, ASG, RDS, CloudWatch, CloudTrail, IAM, KMS, Route 53, S3, and SNS to build a secure, scalable, and highly available AWS environment.
